# Movili Premium SEO

Estrutura entregue:
- Landing page principal (`index.html`)
- 5 páginas satélite em `/servicos`
- CSS global premium e clean
- JS para menu mobile e CTA via WhatsApp
- `robots.txt`
- `sitemap.xml`

## Ajustes antes de publicar
1. Atualize o número do WhatsApp em `assets/js/main.js`
2. Substitua `https://movili.net/` se usar outro domínio
3. Adicione imagens reais em `assets/img` e converta para WebP
4. Conecte GA4, GTM, Search Console e pixels
5. Considere expandir com blog e cases para fortalecer o cluster SEO

## Estrutura SEO
- LP principal para entidade da marca + visão integrada
- páginas satélite para intenção comercial específica
- heading hierarchy
- schema markup
- canonical, OG, robots e sitemap
- interlinking interno entre home e serviços
